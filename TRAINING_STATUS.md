## How to Train the Model

To reproduce or continue training:

```bash

pip install -r requirements.txt

# Download dataset (if not already available)
# Place dataset in data/brain_mri_data/ with subdirectories for each class

# Run training
python train.py --data_dir data/brain_mri_data --epochs_stage1 10 --epochs_stage2 10

# Or for a quick smoke test:
python train.py --data_dir data/brain_mri_data --epochs_stage1 2 --epochs_stage2 2
```

## Training Configuration

The current training configuration is optimized for CPU training:

- **Stage 1 (Feature Extraction)**: 10 epochs with frozen base model
- **Stage 2 (Fine-tuning)**: 10 epochs with unfrozen top layers
- **Batch Size**: 32
- **Image Size**: 224x224
- **Base Model**: MobileNetV2 (pre-trained on ImageNet)
- **Optimizer**: Adam
- **Loss**: Categorical Cross-entropy
